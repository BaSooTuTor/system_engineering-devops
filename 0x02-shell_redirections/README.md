# 0x02. Shell, I/O Redirections and Filters

This directory contains shell script tasks that focus on input/output redirections, filters, and command chaining in Linux. The goal is to build a strong foundation in shell scripting and command-line text processing.

## 📚 Topics Covered

- Standard output and error redirections
- Creating and displaying files using `echo`, `cat`, `head`, `tail`
- Appending to files with `>>`
- Redirecting errors using `2>` and `2>&1`
- Piping commands using `|`
- Using filters like `grep`, `sort`, `uniq`, `wc`, `cut`

## 🛠️ Example Tasks

| Script Name         | Description                                      |
|---------------------|--------------------------------------------------|
| `0-hello_world`      | Prints “Hello, World” to standard output         |
| `1-confused_smiley`  | Displays a confused smiley `"(Ôo)'`             |
| `2-hellofile`        | Displays content of a file                      |
| `3-twofiles`         | Displays contents of two files                  |
| `4-lastlines`        | Displays the last 10 lines of a file            |
| `5-firstlines`       | Displays the first 10 lines of a file           |
| ...                  | ... and more practice scripts                   |
